### Project setup

```text
1. Clone project from the github repository -> "https://github.com/Avirekoka/user-management-system.git" using -> "git clone <repo url>" command

2. To install all npm packages use:
        At package.json run -> "npm install" command

3. To start project use -> "npm start" command

4. open "http://localhost:3000" link

Note:
        At the time of login and registration please put credentials from the reqres.in.

        Eg. Credentials -> email : eve.holt@reqres.in, password : pistol
```
